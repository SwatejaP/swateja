1.Challenges in learning/understanding the concepts

1. Time Complexity Analysis

-Understanding the nature of Big O notation.

-Understanding its practical implications and setting a recursive function in each case of efficiency analysis

2. Binary Search Tree (BST)

- Keeping the tree balanced can be tricky sometimes.

- Implementing operations like insertion and deletion while maintaining BST properties requires careful consideration.

3. Depth-First Search (DFS) and Breadth-First Search (BFS)

- Understanding the differences between DFS and BFS (its traversal method) and when to use each.

- Understanding and implementing backtracking.

4. Heap

- Understanding the properties of a heap and how it differs from other tree structures.

- Implementing insertion, deletion, and heapify operations can be difficult.

5. Sorting

- Performing efficiency analysis and then comparing the time and space complexity of different sorting algorithms.

- Understanding when and how to optimize which sorting algorithm for different cases.

- Understanding the case studies and deciding which sorting method can be best suited for the given case study is more challenging.

6. Pattern Searching

- Analyzing the time complexity for different pattern searching algorithms.

- Implementing these algorithms efficiently.

7. Graph Algorithms

- Dealing with the complexity of graph algorithms, especially for large graphs.

- Understanding the principle of each algorithm.
